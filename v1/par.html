Green Smart Home Management System (Sep-Nov 2022)
  • Project Description
  • The Green Smart Home project is a smart home management system with a microservices architecture. It uses Docker for deployment, Hazelcast for real-time distributed data management, and the MVC pattern for the web service. Data from LoRaWAN sensors is collected through an MQTT broker integrated into the "The Things network stack". This data is then used as the basis for controlling physical devices based on the collected data.
